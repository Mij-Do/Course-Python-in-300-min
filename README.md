## üß† Python Syntax vs Other Languages

Python emphasizes **readability** and simplicity:

- Uses **new lines** to end statements (no semicolons needed).  
- Uses **indentation (whitespace)** to define blocks like loops, functions, and classes (no `{}` curly braces).  
- Designed to feel similar to writing in plain English.
